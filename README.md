# 项目名称：whereNet(仿去哪儿网)

## 项目简介：
&nbsp;&nbsp;&nbsp;&nbsp;这个小项目是用vue来模仿去哪儿网部分页面，数据来源是mock数据。前段时间随着vue3.0 beta版的发布，这个项目已经用vue3.0的Composition API 新特性改造了一下，由于在vue3.0中未找到关于keep-alive的两个生命周期函数，所以keep-alive的特性并没有发挥出来，代码已经提交到new-version分支上。
